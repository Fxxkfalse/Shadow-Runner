# Shadow Runner ― 影だけで世界を読むアクションパズル

ブラウザで動くプロトタイプ実装です。表示されるのは「影」だけ。
プレイヤーは影から本当の地形・敵の挙動を推理して進みます。

操作
- 左右移動: ← / →
- ジャンプ: スペース
- 画面下のボタン: リトライ / 次のステージ

ステージ
1. 静止した光源（チュートリアル）
2. ゆっくり動く光源（影の伸び縮みに注目）
3. 敵の影（影と本体がズレる）
4. 複数光源（二重影）

公開方法
- このリポジトリを GitHub に push し、GitHub Pages を有効にすると Web 上で遊べます（静的な index.html なので簡単です）。
